# UI 契约：公共 Modal 弹窗

## 1. 公共入口

### `openModal(component, props)`

**职责**：以命令式方式打开一个任意组件弹窗。  
**返回值**：`Promise<ModalResult>`

#### 输入约束

| 字段 | 必填 | 说明 |
|------|------|------|
| `component` | 是 | 目标弹窗组件 |
| `props` | 否 | 传递给该组件的入参对象 |

#### 行为约束

1. 调用后必须立即返回 Promise
2. 当前若没有激活中的命令式弹窗，则立即展示当前组件
3. 当前若已有激活中的命令式弹窗，则本次请求必须等待
4. 组件在确认路径下可返回结构化数据
5. 组件在取消或关闭路径下可不返回业务数据

### `openModal(modalEnum, props)`

**职责**：以命令式方式打开组件包内置弹窗。  
**返回值**：`Promise<ModalResult>`

#### 输入约束

| 字段 | 必填 | 说明 |
|------|------|------|
| `modalEnum` | 是 | 组件包内置弹窗标识 |
| `props` | 是 | 与该内置弹窗约定一致的入参 |

#### 行为约束

1. 调用方不需要关心内置弹窗的真实组件路径
2. 不同内置标识必须映射到稳定的组件实现
3. 内置标识路径与任意组件路径共用同一串行队列与结果 Promise 能力

## 2. 基础弹窗契约

### `BaseModal`

**职责**：承接所有弹窗共享的展示与交互能力。

#### 共用能力

| 能力 | 说明 |
|------|------|
| 标题与正文容器 | 统一内容区结构 |
| 按钮文案 | 支持确认与取消文案覆盖 |
| 尺寸与宽度 | 支持预设尺寸与单独宽度 |
| 位置 | 支持是否垂直居中 |
| 按钮状态 | 支持确认按钮未就绪态与附加按钮属性 |
| 异步确认/取消 | 确认或取消动作支持异步执行，成功完成后才结束当前弹窗 |

#### 交互约束

1. 业务弹窗只负责内容与结果，不重复定义基础壳层结构
2. 业务弹窗可以通过共用属性调整标题、按钮和尺寸
3. `onOk` 与 `onCancel` 允许返回同步结果或 Promise，只有成功完成后才关闭弹窗
4. 若 `onOk` 或 `onCancel` 抛错，或返回 rejected Promise，弹窗必须保持打开
5. 异步执行期间需要进入 loading 态，并阻止重复触发同一动作
6. 业务弹窗关闭后必须能触发清理流程

## 3. 注入契约

### `ModalInjectedProps`

**职责**：向命令式弹窗组件提供统一的结束能力。

| 字段 | 说明 |
|------|------|
| `onOk` | 用于确认当前弹窗，并可附带业务结果；允许同步或异步完成 |
| `onCancel` | 用于取消当前弹窗，并可附带业务结果；允许同步或异步完成 |
| `afterClose` | 用于当前弹窗关闭后的清理 |

#### 规则

1. 命令式弹窗组件必须通过注入回调结束本次调用
2. 页面直接挂载组件时，由页面自己提供对应回调
3. 注入回调若为异步动作，只有成功完成后才能关闭当前弹窗
4. 注入回调若执行失败，必须保留当前弹窗并结束 loading

## 4. 内置示例弹窗契约

### `DemoActionModal`

**职责**：作为组件包首个内置示例弹窗，验证“说明文案 + 备注输入 + 结构化结果回传”。

#### 输入约束

| 字段 | 说明 |
|------|------|
| `description` | 展示说明文案 |
| `initialRemark` | 默认备注 |
| `title / okText / cancelText / width / centered / size` | 继承基础弹窗共用配置 |

#### 输出约束

确认结果至少应包含：

- `ticketId`
- `status`
- `operatorName`
- `remark`
- `processedAt`

## 5. 队列契约

| 规则 | 说明 |
|------|------|
| 单实例 | 同一时刻只能存在 1 个激活中的命令式弹窗 |
| 顺序 | 后续请求按触发顺序依次展示 |
| 失败兜底 | 上一个请求即使走取消或关闭路径，也不能阻塞后续请求 |
| 清理 | 当前实例关闭后必须销毁挂载实例与临时容器 |

## 6. 测试页契约

`/Modal` 页面至少包含以下验证内容：

1. 2 个应用侧本地命令式弹窗
2. 1 个组件包内置命令式弹窗
3. 1 个组件包组件直接挂载用例
4. 1 个串行队列验证按钮
5. 1 个最近结果日志区

## 7. 中文化契约

- 标题、说明文案、按钮文案和日志摘要使用简体中文
- 内置示例弹窗返回的结构化信息可被页面直接转成中文日志
- 测试页文案需要能直接服务手工验收
